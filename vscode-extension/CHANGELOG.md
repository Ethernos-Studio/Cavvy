# Change Log

## [0.2.1] - 2026-02-11

### 修复
- 修复预处理器指令误报缺少分号的问题
  - `#define`, `#ifdef`, `#ifndef`, `#endif`, `#undef` 等预处理器指令不再提示缺少分号

## [0.2.0] - 2026-02-11

### 新增
- 预处理器指令支持
  - 语法高亮：`#define`, `#ifdef`, `#ifndef`, `#endif`, `#undef`
  - 代码补全：预处理器指令智能提示
  - Hover 文档：预处理器指令悬停提示
- 增强的代码片段
  - 新增 `fori` - 数组遍历 for 循环
  - 新增 `dowhile` - do-while 循环
  - 新增 `method-varargs` - 可变参数方法
  - 新增 `newarray2d` - 二维数组创建
  - 新增 `array-init` - 数组初始化
  - 新增 `ifdef`, `ifndef`, `define` - 预处理器指令片段
  - 新增 `final-var` - final 常量定义
  - 新增 `static-field` - 静态字段定义
  - 新增 `cast` - 类型转换
- 增强的语言配置
  - JSDoc 风格注释自动格式化
  - 改进的缩进规则
- 扩展关键字支持
  - 新增 `extends`, `implements`, `interface`, `enum`
- 内置函数文档增强
  - 新增 `readInt`, `readFloat`, `readLine` 文档

### 改进
- 语法高亮文件优化
  - 添加可变参数操作符 `...` 高亮
  - 改进数字字面量匹配
  - 优化方法声明匹配
- 代码补全优化
  - 行首自动提示预处理器指令
  - 更智能的代码片段排序
- Hover 文档优化
  - 添加预处理器指令检测
  - 扩展关键字文档

## [0.1.0] - 2026-02-09

### 新增
- 语法高亮支持
  - 关键字、类型、修饰符高亮
  - 字符串、字符、数字字面量高亮
  - 注释高亮
  - 方法调用和字段访问高亮
  - 注解高亮
- 跳转到定义功能 (Ctrl+Click / F12)
  - 支持类、方法、字段、变量、参数
- 语法错误诊断
  - 实时语法检查
  - 括号匹配检查
  - 基本语法规则验证
  - 支持集成 eolc 编译器
- 文档大纲视图支持
  - 类、方法、字段、变量层级展示
- 配置选项
  - 编译器路径配置
  - 诊断开关
  - 诊断延迟配置
